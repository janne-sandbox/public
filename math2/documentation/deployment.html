Chapter 06.1 · Operations

Configure the boundary, cache, and container.

Defaults favor local development: loopback HTTP, a persistent file cache, bounded input, and no H2 console.

Persistent or cache-free

# Persistent H2 cache
MATH2_CACHE_ENABLED=true ./run-dev-local.sh

# No datasource, Hibernate repository, or Liquibase initialization
MATH2_CACHE_ENABLED=false ./run-dev-local.sh

The schema is managed by Liquibase and Hibernate validates it. Result data is CLOB-backed, and canonical-expression plus access-order indexes support cache behavior.

Docker

docker build -f java-rest-service/Dockerfile -t math2:local .
docker run --rm \
  -p 127.0.0.1:8080:8080 \
  -e MATH2_MAX_LENGTH=5000 \
  -e MATH2_CACHE_MAX_ENTRIES=1000 \
  -v math2-data:/app/data \
  math2:local

The image runs as a non-root user. Keep the host mapping on loopback unless a separate authenticated and encrypted ingress is intentionally configured.
