Chapter 05.4 · .NET

Async REST calculation for .NET 8.

The client supports cancellation, configurable timeout/base URL, and injection of an application-owned HttpClient.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Calculate asynchronously

using Nologo2.Math2;

using var client = new Math2Client(timeout: TimeSpan.FromSeconds(30));
var equation = Equation.Parse("sqrt(2)+pi");
Number result = await client.CalculateAsync(
    equation,
    precision: 1000,
    cancellationToken);

Batch

var results = await client.CalculateBatchAsync(
    new[] { Equation.Parse("1/7"), Equation.Parse("pi") },
    precision: 1000);

Error taxonomy

Math2ValidationException, Math2TransportException, Math2ProtocolException, and Math2ServiceException separate caller, network, response, and server failures.
