Chapter 05.6 · Rust

A Rust 2021 blocking client over rustls.

The crate uses Result throughout and represents service failures as a structured enum variant.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Usage

use nologo2_math2::{Client, Equation};

let client = Client::localhost()?;
let equation = Equation::parse("sqrt(2)+pi")?;
let result = client.calculate(&equation, Some(1000))?;
println!("{}", result.as_str());

Batch

let equations = vec![
    Equation::parse("1/7")?,
    Equation::parse("pi")?
];
let results = client.calculate_batch(&equations, Some(1000))?;

Transport and errors

reqwest provides blocking HTTP with rustls. Math2Error distinguishes Validation, Transport, Protocol, and structured Service failures.
