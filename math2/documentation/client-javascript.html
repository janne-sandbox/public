Chapter 05.2 · JavaScript

A zero-dependency ES module client.

Use Node.js 18+ or a browser with fetch. Local values and equations are immutable; calculation happens over REST.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Single calculation

import { Equation, Math2Client, Number } from "@nologo2/math2";

const two = Equation.of(Number.fromInteger(2));
const equation = two.power(Equation.of(Number.fromInteger(1000)));
const result = await new Math2Client().calculate(equation, 1000);
console.log(result.toString());

Batch and client options

const client = new Math2Client({
  baseUrl: "http://127.0.0.1:8080",
  timeoutMs: 30_000,
  fetchImpl: globalThis.fetch
});
const results = await client.calculateBatch([
  Equation.parse("sqrt(2)"),
  Equation.parse("pi")
], 1000);

Errors

ValidationError is local, TransportError covers network/timeouts, ProtocolError rejects malformed success data, and ServiceError carries HTTP status, stable code, message, and optional parser position.
