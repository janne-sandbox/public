Chapter 05.5 · C++

C++20 values with a libcurl transport.

The native API uses immutable-style value returns and keeps every decimal in std::string.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Usage

#include "math2/math2.hpp"
using namespace nologo2::math2;

auto equation = Equation::parse("sqrt(2)+pi");
auto result = Client("http://127.0.0.1:8080", 30000)
    .calculate(equation, 1000);
std::cout << result.str() << '\n';

Build and test

cmake -S cpp-api -B cpp-api/build
cmake --build cpp-api/build
ctest --test-dir cpp-api/build

Failures

Catch ValidationError, TransportError, ProtocolError, or ServiceError. The service error provides HTTP status, stable code, and optional expression position.
