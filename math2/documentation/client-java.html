Chapter 05.1 · Java

Calculate directly in the JVM.

The Java library is the authoritative implementation and does not require HTTP, Spring, or H2.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Maven dependency

<dependency>
  <groupId>com.nologo2</groupId>
  <artifactId>math2-java-api</artifactId>
  <version>0.1.0-SNAPSHOT</version>
</dependency>

Install the reactor into the local Maven repository with mvn install until a release repository is configured.

Fluent equation

Number radius = Number.fromString("10");
Equation area = radius.power(Number.fromInteger(2))
    .multiply(Number.pi());
Number result = area.calculate(1000);

Parsed equation

Equation equation = Equation.parse("pi*10^2");
Number result = equation.calculate(1000);

Conversions

toInt, toLong, toFloat, and toDouble only succeed for an exact round trip. Otherwise PrecisionLossException exposes the source, target type, and nearest primitive value.
