Chapter 05.3 · TypeScript

The same REST client with static contracts.

Typed values, client options, service errors, and readonly batch results make the protocol visible to the compiler.

LibrariesOverviewJavaJavaScriptTypeScript.NETC++Rust

Typed usage

import { Equation, Math2Client, Number } from "@nologo2/math2-typescript";

const equation: Equation = Equation.of(Number.fromInteger(9))
  .sqrt()
  .add(Equation.parse("2^3"));

const result: Number = await new Math2Client().calculate(equation, 1000);

Custom transport

const client = new Math2Client({
  baseUrl: "http://127.0.0.1:8080",
  timeoutMs: 15_000,
  fetchImpl: fetch
});

The source is exported directly as TypeScript. The repository tests run on Node.js 22.6+:

cd ts-api && npm test
