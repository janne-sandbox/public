Code Review System - C# Agent

Scope

The C# agent scans *.cs source files and reads *.csproj project files to check nullable-reference configuration.

LANGUAGE=csharp ./run-review.sh # Aliases LANGUAGE=c# ./run-review.sh LANGUAGE=cs ./run-review.sh

Deterministic Checks

  • async void methods, which should normally return Task.
  • Blocking asynchronous work through .Result or .Wait().
  • SQL commands constructed through interpolation or concatenation.
  • Missing <Nullable>enable</Nullable> project configuration.
  • Missing test/spec files or directories.
  • Evidence for the first five language-relevant functional requirements.

These checks always run and do not require the .NET SDK.

Deepseek Review

When enabled and available, Deepseek filters the requirements and reviews bounded code samples for correctness, security, resource management, and maintainability.

# Offline through local Ollama LANGUAGE=csharp INTERNET=false DEEPSEEK=true ./run-review.sh # Online through the hosted API export DEEPSEEK_API_KEY='your-key' LANGUAGE=csharp INTERNET=true DEEPSEEK=true ./run-review.sh
Online data boundary: online mode sends requirement headings and sampled C# source to the hosted Deepseek API. Use offline or deterministic mode for code that must remain local.

Limitations

  • The agent does not run Roslyn analyzers, dotnet build, tests, or package vulnerability checks.
  • async void event handlers can be legitimate and may require manual dismissal.
  • Deepseek findings are advisory until confirmed by a developer.
