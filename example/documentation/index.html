Example Project Documentation

Review Project Changes Locally

Automated 4 Languages Local-First Requirements-Driven

The review tool scans Java, Rust, React, and TypeScript code against the project's functional and non-functional requirements. Each language-specific agent produces a Markdown report with findings, suggestions, and a final verdict.

Default behavior: all four agents run without internet access. Deepseek analysis is enabled when a local Ollama service is available and otherwise falls back to deterministic checks.

Quick Start

Run every review agent

./run-review.sh

Review one language

LANGUAGE=java ./run-review.sh LANGUAGE=typescript ./run-review.sh

Use deterministic checks only

DEEPSEEK=false ./run-review.sh

Reports are written to review/results/<timestamp>/. Set LANGUAGE to java, rust, react, typescript, or all.
