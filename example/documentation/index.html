Example Project Documentation

Review Project Changes Locally

Automated 6 Languages Local-First Requirements-Driven

The review tool scans Java, Rust, React, TypeScript, C++, and C# code against the project's functional and non-functional requirements. Each language-specific agent produces a Markdown report with findings, suggestions, and a final verdict.

Default behavior: all six agents run offline. Deepseek uses local Ollama when available and otherwise falls back to deterministic checks. With INTERNET=true, Deepseek uses its hosted API and requires DEEPSEEK_API_KEY.

Quick Start

Run every review agent

./run-review.sh

Review one language

LANGUAGE=java ./run-review.sh LANGUAGE=typescript ./run-review.sh LANGUAGE=cpp ./run-review.sh LANGUAGE=csharp ./run-review.sh

Use deterministic checks only

DEEPSEEK=false ./run-review.sh

Reports are written to review/results/<timestamp>/. Set LANGUAGE to java, rust, react, typescript, cpp, csharp, or all.
